# Shark Tank India Dashboard

## Project Overview
This Tableau dashboard analyzes the Shark Tank India dataset to identify investment trends, brand performance, and shark-wise deal statistics.

## Dashboard Includes
- Top Deals by Brand
- Deals by Shark
- Deal Status
- Top 10 Highest Amount Per Shark
- Valuation vs Deal

## Tools Used
- Tableau Public
- CSV Dataset

## Dataset
Shark Tank India Dataset
